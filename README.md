# Bus-guru-Proj
Bus Guru is a smart urban commute application featuring real-time bus tracking, QR-based e-ticketing, and AI-powered commuter assistance. Developed using Flutter and a Flask backend, the project demonstrates core smart transport workflows. Screenshots included show preserved frontend modules after partial data loss.
During development, the system experienced an unexpected PC failure which resulted in partial loss of frontend source files. However, the core architecture, backend services, and key frontend modules were already implemented and tested.

The uploaded screenshots serve as visual proof of the implemented UI flows, including login, dashboard, bus tracking, ticket booking, QR ticket generation, chatbot interaction, and conductor validation. The project remains a valid demonstration of a smart urban transport system.
